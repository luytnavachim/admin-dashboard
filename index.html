Facturen verwerken

Kies een factuur-mail uit Inbox. Het dashboard: (1) leest de PDF (en eventuele UBL-bijlage), (2) extraheert leverancier + bedragen via Claude, (3) maakt/koppelt de relation in Informer, (4) maakt de inkoopfactuur aan met PDF als bijlage, (5) verplaatst de mail naar MoneyMonk Verwerkt. Negeren verplaatst de mail naar Archive zonder factuur aan te maken.

Klik "Verversen" om mails op te halen.

Losse PDF uploaden

Heb je een factuur-PDF die niet in je inbox zit (bv. download uit een portaal of scan)? Upload 'm hier — zelfde extractie + Informer-flow, zonder Outlook-stappen.

Informer-verbinding

Werkt via een Cloudflare Worker. Informer staat geen directe browser-calls toe (CORS), dus we routeren via een Worker die de API-key server-side bewaart. Setup-stappen staan in worker.js; deploy via de "Push" knop in je Deploy Dashboard.

Verbinding testen

Doet een GET op het opgegeven endpoint. Default /administration/ — geeft details van de administratie die bij je API-key hoort.
